# 📚 Pahana Education Billing System - Frontend

A modern **React.js** frontend application for the **Pahana Education Billing System**. This web application provides an intuitive interface for managing customers, books, inventory, and billing operations for Pahana Education Bookshop.

---

## 🚀 Features

- 🔐 User Authentication
  - Secure Login
  - Protected Routes

- 📊 Dashboard
  - Overview of system statistics
  - Quick access to major modules

- 👤 Customer Management
  - Add new customers
  - View customer list
  - Edit customer information
  - Delete customers
  - Search customers

- 📚 Book & Inventory Management
  - Add new books
  - Update book details
  - Delete books
  - Search inventory
  - View stock availability

- 🧾 Billing Management
  - Create customer invoices
  - Add multiple items
  - Automatic total calculation
  - Generate bills

- 📱 Responsive Design
  - Mobile-friendly UI
  - Modern and clean interface

---

## ⚙️ Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Clone the repository

```bash
git clone https://github.com/your-username/pahana-education-billing-system-frontend.git
```

### Navigate to the project

```bash
cd pahana-education-billing-system-frontend
```

### Install dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the project root.

```env
VITE_API_URL=http://localhost:8080/api
```

Update the API URL according to your backend server.

### Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

## 🔗 Backend

This frontend communicates with the Java backend using REST APIs.

Make sure the backend server is running before starting the frontend.
